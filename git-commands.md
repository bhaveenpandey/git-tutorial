# Repo initilazition

git init

# Do the file changes

# add for tracking
```
git add .
git add <filename>
```

# do the commmit
```
git commit -m "title"
git commit -m "title" -m "description"
```

# change the branch name to main
```
git branch -M main
```

# add the origin repo
```
git remote add origin <github repo url>
```

# push to repo
```
git push -u origin main
```
